# 中医诊断学-切诊-脉诊1

## 一、主要概念

### 1. 脉诊的定义与意义
- 脉诊是中医四诊（望闻问切）之一，通过触摸脉搏的变化来诊断疾病
- 脉诊部位：**独取寸口**（桡侧桡动脉处）
- 原理：寸口脉为脉之大会，脏腑气血变化可反映于此

### 2. 脉诊部位的历史演变
| 时期 | 诊脉部位 |
|------|----------|
| 《黄帝内经》 | 三部九候（头面、上肢、下肢各分上中下） |
| 《伤寒论》张仲景 | 三部：人迎脉（颈动脉）、寸口脉（桡动脉）、趺阳脉（足背动脉） |
| 现代临床 | 独取寸口 |

---

## 二、详细内容

### 1. 寸口脉的定位（寸关尺）

```
        左手                    右手
    ┌───┬───┬───┐          ┌───┬───┬───┐
    │寸 │关 │尺 │          │寸 │关 │尺 │
    └───┴───┴───┘          └───┴───┴───┘
     ↑     ↑     ↑           ↑     ↑     ↑
    心    肝胆  肾阴         肺   脾胃  肾阳
```

- **左手**：心 → 肝胆 → 肾阴
- **右手**：肺 → 脾胃 → 肾阳
- 临床扩展：肾阳可察大便情况，肾阴可察小便情形

### 2. 诊脉手法：举按循

| 手法 | 含义 | 作用 |
|------|------|------|
| **举** | 轻取 | 轻触皮肤，感知浮脉 |
| **按** | 重取 | 用力深按，感知沉脉 |
| **循** | 中取/寻找 | 调整力量寻找脉位，可上下左右前后移动 |

### 3. 脉位：浮中沉

- **浮**：轻取即得，重按稍减
- **中**：不轻不重，中等力度
- **沉**：轻取不应，重按始得

### 4. 脉诊要素总结

```
脉诊 = 部位（寸关尺）× 指力（举按循）× 脉位（浮中沉）
      = 三部九候（后世发展）
```

> **难点提示**：后世"三部九候"与《黄帝内经》"三部九候"含义不同，不可混淆。

---

## 三、重点总结

1. **核心考点**：独取寸口法，左右手寸关尺对应不同脏腑
2. **记忆口诀**：左心肝肾阴，右肺脾胃阳
3. **临床要点**：诊脉时需结合举、按、循三种手法，体会浮、中、沉不同脉位
4. **扩展知识**：讲师提及个人临床使用"五层脉法"更精确，后续课程详述

## 四、临床提示

- 脉诊需反复实践，体会指下感觉
- 脏腑对应关系为基本框架，临床需灵活运用
- 肾阳（尺部）主大便，肾阴（尺部）主小便，为经验之谈
