# 中医诊断学-问诊-十问

## 一、问诊概述

### 1. 问诊的定义
问诊是中医四诊（望、闻、问、切）之一，通过医生与病人的直接对话，询问病人的症状、病史等相关信息进行诊断的方法。

### 2. 问诊的重要性
- **最常用的诊断方法**：在临床实践中应用最为广泛
- **获取关键信息**：病人往往不会主动提供重要的诊断信息
- **体现医生水平**：高明的医生只需询问一两个关键问题即可掌握病情要点

> **重点**：问诊是一项技术性很强的诊断方法，医生必须掌握系统的问诊技巧。

## 二、十问歌详解

### 1. 张景岳《景岳全书》十问

| 序号 | 问诊内容 | 诊断意义 |
|:---:|:---|:---|
| 一 | 问寒热 | 辨别表证、寒热属性 |
| 二 | 问汗 | 了解表邪情况及阳气盛衰 |
| 三 | 问头身 | 排查疼痛、不适等症 |
| 四 | 问便 | 审查大小便情况（脾胃、大肠功能） |
| 五 | 问饮食 | 判断脾胃功能、食欲状况 |
| 六 | 问胸 | 探查胸胁苦满（少阳病证） |
| 七 | 问耳聋 | 了解听力及肾气状况 |
| 八 | 问口渴 | 辨别津液损伤程度 |
| 九 | 问脉色 | 察阴阳（结合切诊与望诊） |
| 十 | 问旧病 | 了解既往病史，判断体质倾向 |

### 2. 陈修园《医学实在易》十问

| 序号 | 问诊内容 | 诊断意义 |
|:---:|:---|:---|
| 一 | 问寒热 | 同上 |
| 二 | 问汗 | 同上 |
| 三 | 问头身 | 同上 |
| 四 | 问便 | 同上 |
| 五 | 问饮食 | 同上 |
| 六 | 问胸 | 同上 |
| 七 | 问耳聋 | 同上 |
| 八 | 问口渴 | 同上 |
| 九 | 问旧病 | 了解既往病史，作为体质参考 |
| 十 | 问病因 | 推断当前发病原因（体现医生功力） |

> **难点**：陈修园十问中"问病因"是最能体现医生专业水平和诊断功力的部分。

## 三、专科补充问诊

### 1. 妇科问诊
- **问经带**：月经和白带情况是妇科诊断的关键
- **内容要点**：
  - 月经周期
  - 月经量
  - 月经颜色
  - 白带情况
- **临床意义**：经带胎产是妇科辨证的重要依据

### 2. 儿科问诊
- **问惊疳**：小儿惊风、疳积等常见病
- **问天花麻疹**：了解患儿传染病史
- **特殊意义**：小儿表达能力有限，需依赖家长叙述和医生详细询问

> **重点**：妇科和儿科问诊是基础十问的重要补充，不同人群需要侧重不同的问诊内容。

---

## 四、重点总结

1. **问诊是四诊中技术性最强的部分**，需要医生主动引导病人提供诊断信息
2. **十问歌是问诊的基础框架**，涵盖寒热、汗、头身、便、饮食、胸、耳聋、口渴、旧病、病因等方面
3. **问病因是问诊的难点**，需要医生具备丰富的临床经验和辨证思维能力
4. **专科问诊需补充**：妇科必问经带，儿科必问惊疳、天花麻疹
5. **临床原则**：病人往往不会主动诉说重要症状，医生必须系统询问，不能仅凭病人主诉

## 五、学习要点

- 熟练掌握十问歌内容
- 理解每项问诊的诊断意义
- 学会根据病人具体情况灵活运用
- 掌握专科问诊的特殊要求
