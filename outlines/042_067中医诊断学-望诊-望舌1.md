# 中医诊断学-望诊-望舌（一）

## 一、舌诊概述

### 1. 舌诊的重要性

- 舌诊是中医诊断中的重要组成部分
- 后世发展资料丰富，论述详尽
- 相比脉诊，舌诊学习较为容易
- 建议投入更多时间精力学习，临床经验丰富后望诊能力将显著提升

---

## 二、舌与脏腑的对应关系

### 1. 舌诊脏腑分布图

| 舌部区域 | 对应脏腑 | 备注 |
|---------|---------|------|
| 舌尖 | 心 | 心火盛则舌红 |
| 舌尖后方 | 肺 | — |
| 舌中部左侧 | 肝 | 左肝 |
| 舌中部右侧 | 胆 | 右胆 |
| 正中间 | 胃 | 脾胃区域中心 |
| 舌边缘两侧 | 脾 | 浅黄色区域 |
| 舌根部 | 肾、膀胱 | 下焦问题表现区 |

### 2. 舌部对应脏腑记忆要点

```
        舌尖 → 心 ← 肺（舌尖后方）
        
        左肝     脾胃    右胆
              （胃在正中）
              （脾在边缘）
        
        舌根 → 肾、膀胱（下焦）
```

> ⚠️ **注意**：图中左右为**患者**之左右，故左肝右胆

---

## 三、学术说明

### 不同学派的差异

- 不同学派对舌部脏腑位置的划分存在差异
- 讲师本门学派与此图分布有所不同
- 各学派均有其独特的观察方法，均有临床价值
- 本图采用最常见的脏腑分布方式

---

## 四、重点总结

1. **舌诊是望诊的重要内容**，需重点学习
2. **舌诊脏腑分布图**是基础，需熟记：
   - 舌尖：心、肺
   - 舌中：肝（左）、胆（右）、胃（正中）、脾（边缘）
   - 舌根：肾、膀胱
3. 临床观察时需注意**患者的左右**与**医师相对的左右**的区别
4. 舌象（颜色、形状等）可反映体内脏腑状况

---

## 五、临床意义

舌诊通过观察舌质、舌苔的变化，可判断：
- 脏腑气血盛衰
- 病邪性质与部位
- 病情进退与预后

是中医辨证论治的重要依据之一。
