# 中医诊断学-切诊-按诊（第一部分）

## 一、概述

### 1. 切诊的分类

- **切诊**分为两部分：
  - **脉诊**：脉象诊断
  - **按诊**：触按诊断（本节重点）

### 2. 按诊的主要内容

本节讲授按诊中的**背诊**部分

---

## 二、背诊的原理

### 1. 理论基础

- 脊柱两侧的**膀胱经第一线、第二线**是周围神经传导至中枢神经的重要通道
- 膀胱经与督脉循行于背部，与全身脏腑密切相关
- 背部俞穴是脏腑之气输注于背部的部位

### 2. 背部与脏腑的对应关系

| 椎体位置 | 俞穴 | 对应脏腑 |
|---------|------|---------|
| 第一椎 | 大杼 | 骨 |
| 第二椎 | 风门 | |
| 第三椎 | 肺俞 | 肺（金） |
| 第四椎 | 厥阴俞 | |
| 第五椎 | 心俞 | 心（火） |
| 第六椎 | 督俞 | |
| 第七椎 | 膈俞 | 膈 |
| 第九椎 | 肝俞 | 肝（木） |
| 第十椎 | 胆俞 | 胆 |
| 第十一椎 | 胃俞 | 脾 |
| 第十二椎 | 胃俞 | 胃 |
| 肾俞 | 肾俞 | 肾（水） |

> **口诀**：一是大杼二风门，三是肺俞四厥阴，心五督俞膈俞七，肝胆脾胃十一十二肾俞

### 3. 对应规律

- 背部俞穴按五行排列：**金、火、木、土、水**
- 前面肺的位置对应第三椎
- 前面心的位置对应第五椎

---

## 三、背诊的操作方法

### 1. 体位

- **患者体位**：俯卧位（趴着）
- **不宜坐位**：坐着不好按压

### 2. 按诊手法

- 使用**尺骨鹰嘴**（肘部）进行按压
- 按压部位：膀胱经第一线、第二线
- **操作顺序**：
  - 前三椎：上下按压
  - 第四椎起：左右按压
- 寻找**痛点**作为诊断依据

---

## 四、临床诊断意义

### 1. 阳性反应

- 按压某处出现**疼痛**，提示对应脏腑可能存在问题

### 2. 辨证要点

> **重点**：需结合问诊综合判断

**临床实例**：

- 按压**第九椎（肝俞）**疼痛
- 伴随症状：
  - 眼睛酸涩
  - 情绪易怒
  - 睡眠不佳
- 综合判断：提示肝的问题

---

## 五、重点总结

1. **背诊原理**：背部膀胱经、督脉与脏腑密切相关，俞穴是脏腑之气输注之处
2. **诊断依据**：按压背部俞穴寻找痛点
3. **辨证关键**：必须结合问诊信息综合判断
4. **操作特点**：简便易行，临床实用性较强

---

## 六、临床应用提示

- 按诊时应注意左右对比
- 痛点程度可提示病变轻重
- 需与患者主诉、其他诊法结果相互印证
