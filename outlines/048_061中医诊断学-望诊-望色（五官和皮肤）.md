# 中医诊断学-望诊-望色（五官和皮肤）

## 一、概述

望诊是中医诊断的重要方法，通过观察患者的面部、五官和皮肤的色泽、形态来推断脏腑气血的变化。面部是人体健康的“镜子”，能够反映内脏的功能状态。

> **重点**：望诊时首先观察患者的面部，这是获取健康资讯最直接、最常用的部位。

---

## 二、五官与脏腑的对应关系

### 1. 面部脏腑分区图

| 面部区域 | 对应脏腑 | 说明 |
|---------|---------|------|
| 两眉之间（印堂） | 肺 | 发红表示肺热 |
| 两目之间 | 心 | 反映心脏状况 |
| 鼻梁正中间（山根） | 肝 | 小儿此处常见，发青表示肝寒 |
| 鼻梁两侧 | 肝胆 | 肝的辅助区 |
| 鼻梁外侧 | 小肠 | 发红表示小肠热 |
| 鼻翼 | 大肠 | 鼻翼两侧为大肠区 |
| 鼻头 | 脾 | 反映脾的功能 |
| 鼻翼两侧 | 胃 | 脾胃中焦区域 |
| 人中（水沟） | 膀胱 | 膀胱面色反应区 |
| 嘴角旁边 | 肾 | 反映肾脏状况 |

### 2. 临床意义

- **印堂发红** → 肺热
- **山根发青** → 肝寒（小儿常见）
- **鼻翼发红** → 大肠热
- **鼻头色变** → 脾病

---

## 三、面色诊断要点

### 1. 青色

- **主病**：肝病、内风、痛症（尤以游走性疼痛为多，如风湿痛）
- **理论依据**：青色属木，肝亦属木，同气相求
- **特点**：整体面部呈现青绿色调

### 2. 黄色

| 类型 | 主病 | 特点 |
|-----|------|------|
| 淡黄有红斑 | 脾虚、干血瘀 | 兼有瘀血表现 |
| 萎黄（暗黄） | 气血两虚 | 面黄肌瘦，色泽晦暗 |
| 亮黄 | 脾病 | 色泽鲜明 |

> **重点**：萎黄者多属气血两虚，需与亮黄鉴别

### 3. 黑色

- **主病**：
  - 阳气不足，肾阳虚
  - 寒湿内盛
  - 血运不畅，瘀血所致
- **特点**：面部晦暗无光

### 4. 白色

- **主病**：营血不荣于面
- **兼病**：虚寒证、血虚证
- **特点**：面色苍白无血色，病态惨白

> **注意**：需与正常的细腻嫩白鉴别

### 5. 红色

- **主病**：体内有热
- **临床应用**：如两目之间发红，提示心有热

---

## 四、预后判断

### 印堂色泽变化

- **病理状态**：两眉间（肺区）呈红色
- **恢复标志**：转为青润黄色 → 胃气恢复，疾病向愈

---

## 五、重点总结

1. **面部分区是望诊基础**：必须熟记五官与脏腑的对应关系
2. **色泽与病性相关**：
   - 青→肝病/风/痛
   - 黄→脾病（需分类型）
   - 黑→阳虚/寒湿/瘀血
   - 白→血虚/寒
   - 红→热
3. **动态观察有意义**：如印堂颜色从红转青黄，提示胃气恢复

---

## 六、临床实例

| 患者表现 | 诊断提示 |
|---------|---------|
| 印堂发红 | 肺热 |
| 山根（鼻梁中）发青 | 肝寒（小儿多见） |
| 面色萎黄 | 气血两虚 |
| 面色黧黑 | 肾阳不足，寒湿内盛 |
| 面色苍白 | 血虚或虚寒 |

---

> **难点**：黄色需仔细鉴别——亮黄与萎黄代表不同的病机，前者偏实，后者偏虚。
