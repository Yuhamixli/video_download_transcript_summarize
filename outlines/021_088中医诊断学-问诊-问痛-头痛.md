# 中医诊断学-问诊-问痛-头痛

## 一、概述

头痛为临床常见症状，引起原因较多，需根据疼痛部位进行辨证论治。中医依据头痛部位将其分为**六经头痛**，不同部位的头痛对应不同的经络、病因、引经药及穴位。

---

## 二、六经头痛的辨证论治

### 1. 后头痛（太阳头痛）

| 项目 | 内容 |
|------|------|
| **疼痛部位** | 后枕部、枕骨下缘 |
| **主要病因** | 外感风寒（感冒）、肾虚 |
| **经络归属** | 足太阳膀胱经（与肾相表里） |
| **引经药** | 羌活（引药上背脊） |
| **穴位** | 后溪、昆仑 |

> **重点**：感冒头痛以后头痛为多，肾虚者亦可导致后头痛。

---

### 2. 前头痛（阳明头痛）

| 项目 | 内容 |
|------|------|
| **疼痛部位** | 前额、眉棱骨 |
| **主要病因** | 脾胃问题、胃火盛、鼻炎/鼻窦炎 |
| **经络归属** | 足阳明胃经 |
| **引经药** | 白芷、葛根 |
| **穴位** | 鼻三针（迎香×2 + 硬腭）、合谷、内庭 |

**辨证要点**：
- 胃热所致：取合谷、内庭（泻胃火）
- 鼻炎所致：配合鼻三针及补肾穴位（太溪、肾俞）

---

### 3. 偏头痛（少阳头痛）

| 项目 | 内容 |
|------|------|
| **疼痛部位** | 两侧太阳穴附近 |
| **主要病因** | 肝血虚、肝阳上亢 |
| **经络归属** | 足少阳胆经 |
| **引经药** | 柴胡、黄芩、川芎 |
| **穴位** | 外关、风池 |

> **难点**：肝阳上亢型偏头痛的诊断要点需重点掌握。

#### 肝阳上亢型偏头痛的诊断特征

- **诱因**：食用酸性食物（如柑橘类）后头痛加剧
- **情绪相关**：情绪忽然放松或兴奋后头痛加重
- **伴随症状**：
  - 长期头痛
  - 肩颈僵直强硬
  - 颠顶或眼球后剧痛
  - **严重症状**：只能坐着睡，躺下难过（肝阳上亢重症）

---

### 4. 满头痛（少阴头痛/肾虚头痛）

| 项目 | 内容 |
|------|------|
| **疼痛部位** | 整个头部 |
| **主要病因** | 外感风寒、肾虚（空痛） |
| **经络归属** | 足少阴肾经 |
| **引经药** | 细辛、独活 |
| **穴位** | 太溪、肾俞 |

**辨证要点**：
- 外感风寒：满头痛，伴感冒症状
- 肾虚：空痛（自觉头脑空空的发疼）

---

### 5. 太阴头痛（湿邪头痛）

| 项目 | 内容 |
|------|------|
| **疼痛部位** | 整个头部，有被包裹感 |
| **主要病因** | 湿邪（太阴湿土） |
| **经络归属** | 足太阴脾经 |
| **引经药** | 苍术、白术（去湿） |
| **穴位** | 丘墟、阴陵泉（去湿） |

---

### 6. 颠顶痛（厥阴头痛）

| 项目 | 内容 |
|------|------|
| **疼痛部位** | 颠顶（百会穴附近）正上方、正中间 |
| **主要病因** | 肝血虚、寒凝肝脉、肝阳上亢 |
| **经络归属** | 足厥阴肝经 |
| **引经药** | 吴茱萸（一般）、藁本（外感风寒） |
| **穴位** | 太冲 |

#### 颠顶痛辨证分型

| 证型 | 特征 | 治法 |
|------|------|------|
| 肝血虚 | 躺下减轻 | 养血柔肝 |
| 寒凝肝脉 | 伴剧烈寒热感 | 温经散寒（吴茱萸汤） |
| 肝阳上亢 | 剧痛 | 平肝潜阳 |

---

## 三、重点总结

1. **头痛问诊要点**：首先需明确疼痛部位，这是辨证的基础
2. **六经头痛对应关系**：
   - 后头痛 → 太阳头痛 → 膀胱经
   - 前额痛 → 阳明头痛 → 胃经
   - 偏头痛 → 少阳头痛 → 胆经
   - 满头痛 → 少阴头痛 → 肾经
   - 头有包裹感 → 太阴头痛 → 脾经
   - 颠顶痛 → 厥阴头痛 → 肝经
3. **引经药的应用**：根据头痛部位选用相应引经药，可使药力直达病所
4. **肝阳上亢的辨识**：偏头痛伴酸食加重、情绪波动加剧、肩颈僵直、眼球后痛、只能坐睡者为肝阳上亢重症

---

## 四、临床要点

- 头痛分部位论治是中医特色，辨证精确则疗效显著
- 问诊时需询问：疼痛部位、诱因、加重/缓解因素、伴随症状
- 引经药与对应穴位配合使用可增强疗效
