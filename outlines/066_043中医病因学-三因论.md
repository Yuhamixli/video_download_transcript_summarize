# 中医病因学-三因论

## 一、课程概述

**病因学意义**：明确病因是治疗的前提，若不明病因则治疗如同缘木求鱼。

**三因论起源**：宋代陈无择（陈言）在《三因极一病证方论》中首次系统提出，将致病因素归纳为三大类。

---

## 二、三因论核心内容

### 1. 内因——七情致病

**定义**：内在情绪因素导致的疾病。

| 七情 | 致病特点 |
|------|----------|
| 喜 | 过喜伤心 |
| 怒 | 怒则气上 |
| 忧 | 忧则气郁 |
| 思 | 思虑伤脾 |
| 悲 | 悲伤过度伤肺 |
| 恐 | 恐惧伤肾 |
| 惊 | 惊则气乱 |

**重点**：中医自古认识到心理与生理不可分割，情绪问题可直接导致生理疾病。

---

### 2. 外因——六淫致病

**定义**：外来邪气侵犯人体所致疾病。

| 六淫 | 基本概念 |
|------|----------|
| 风 | 风为阳邪，其性开泄 |
| 寒 | 寒为阴邪，易伤阳气 |
| 暑 | 暑为阳邪，多挟湿邪 |
| 湿 | 湿为阴邪，重浊黏滞 |
| 燥 | 燥伤津液 |
| 火 | 火为阳邪，易生肿毒 |

**临床意义**：许多疾病的根本原因可追溯至六淫外感。

---

### 3. 不内不外因

**定义**：既非七情内伤，亦非六淫外感的其他致病因素。

**主要类别**：

- **饮食因素**：过饱、过饥、饮食不节
- **劳倦因素**：过度劳累、体力透支
- **外伤因素**：跌打损伤、摔伤、碾压、溺水
- **虫兽伤害**：蜂蜇、蚊虫咬伤
- **其他**：意外事故等

---

## 三、重点总结

1. **三因分类框架**：
   - 内因：七情（喜怒忧思悲恐惊）
   - 外因：六淫（风寒暑湿燥火）
   - 不内不外因：饮食、劳倦、外伤等

2. **核心思想**：三因论将复杂病因归类为最基本、最基础的因素，便于临床辨证论治。

3. **临床意义**：明确病因分类有助于追溯疾病根源，指导针对性治疗。

---

**【难点提示】**
六淫与不内不外因的界定：蚊虫叮咬虽为外邪，但因其不属于风寒暑湿燥火范畴，故归入不内不外因。
