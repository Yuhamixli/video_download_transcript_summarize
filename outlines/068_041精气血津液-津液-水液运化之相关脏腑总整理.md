# 津液——水液运化之相关脏腑总整理

## 一、津液的基本概念

津液在中医理论中指人体正常的液体物质，有时亦称**“水液”**。其运化代谢涉及多个脏腑的协调配合。

---

## 二、与水液运化相关的脏腑

### 1. 脾

- **脾主湿而恶湿**，为调节水液的核心脏腑
- 初步将水分成**纯水**与**不纯的水**
- 负责水液的输布与分配

### 2. 肺

- **肺主肃降，通调水道**
- 调节水液运行：清的水液**向上**输布，浊的水液**向下**传导
- 通过呼吸参与水液代谢

### 3. 肾

- **肾为水脏**，是水液代谢的根本
- 肾具有**分开输布**水液的功能
- 调节全身水液平衡

> **【重点】** 肾为水液代谢的核心脏腑

### 4. 三焦

- **三焦有两个定义**：
  - 上焦、中焦、下焦（三焦分部）
  - **油网**（即现代医学的**淋巴系统**）
- 淋巴系统作为血液系统之外的体液循环管道，全程参与水液代谢
- 负责脏腑之间的水液传输

### 5. 胃

- **胃为津液来源**
- 饮水首先进入胃部
- 由胃初步受纳后，再由脾进行后续处理

### 6. 小肠

- **小肠主“分清别浊”**
- 消化后的物质经小肠分离：
  - **清者**（有用物质）→ 向上输送至肺、肝、血液
  - **浊者**（废物）→ 向下输送至大肠
- 小肠在短时间内完成水液的清浊分辨

---

## 三、水液运化流程概述

```
饮水 → 胃 → 脾（分纯与不纯）→ 小肠（分清别浊）
                ↓
            肺（通调水道）
                ↓
            肾（水脏调节）
                ↓
            三焦/淋巴（传输分布）
```

---

## 四、重点总结

| 脏腑 | 主要功能 |
|------|----------|
| 脾 | 调节水液，分离纯浊 |
| 肺 | 通调水道，肃降水液 |
| 肾 | 水液代谢的根本，分开输布 |
| 三焦 | 油网/淋巴系统，水液传输通道 |
| 胃 | 津液来源，受纳水液 |
| 小肠 | 分清别浊，辨别水液清浊 |
